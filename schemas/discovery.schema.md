# Discovery Report Schema

Discovery reports are generated by the `exploratory-tester` skill after exploring a live application.

## File Naming

`discovery-<scope>.report.md` — where `<scope>` is the exploration target (e.g., `discovery-login.report.md`, `discovery-checkout.report.md`)

**Location:** `.argus/reports/discovery/`

## Frontmatter

```yaml
---
id: discovery-<scope>
scope: <exploration target description>
url: <entry point URL>
timestamp: <ISO 8601>
auth_method: <form | cookie | token | none>
pages_discovered: <number>
max_depth_reached: <number>
status: <complete | partial | failed>
---
```

## Sections

### 1. Summary
Overview of exploration session: scope, entry point, pages discovered, depth reached, status.

### 2. Pages Discovered
Table with columns:
| # | Page | URL | Screenshot | Notes |
Where screenshot is a relative path to the captured image.

### 3. UI Elements Found
Table per page with columns:
| Element | Selector | Type | Interactive | Notes |
Types: button, input, link, dropdown, checkbox, radio, modal, etc.

### 4. API Calls Observed
Table with columns:
| Method | Endpoint | Status | Response Type | Notes |
Only first-party API calls (third-party scripts filtered out).

### 5. Behaviors Noted
Bullet list of observed interactions, state changes, transitions, animations.

### 6. Destructive Actions Found
Table with columns:
| Action | Location | Why Flagged |
Actions the agent did NOT click (delete, submit payment, form submissions that modify data).
Flagged as "needs manual exploration."

### 7. Questions for Tester
Bullet list of undocumented features, ambiguous behaviors, potential issues the tester should investigate.

## Exploration Bounds

| Setting | Default | Config Key |
|---------|---------|------------|
| Max pages | 20 | `exploration.maxPages` |
| Max depth | 5 | `exploration.maxDepth` |
| Wait strategy | Network idle + 1s DOM stability | `exploration.waitStrategy` |
| Destructive action policy | Never click, flag in report | — |
| SPA handling | Detect client-side routing as new pages | — |
