# UI Verification Report Schema

UI verification reports are generated by the `ui-verifier` skill when comparing design images against live implementations.

## File Naming

`<feature-id>.ui-verification.md`

**Location:** `.argus/reports/ui-verification/`

## Frontmatter

```yaml
---
id: <feature-id>-ui-verification
feature: <feature-id>
design_source: <path or description of design image>
implementation_url: <URL or screenshot path>
timestamp: <ISO 8601>
viewport: <width>x<height>
overall_match: <percentage>
status: <pass | fail | partial>
---
```

## Sections

### 1. Summary
Overall match percentage, viewport tested, design source description.

### 2. Element Presence Check
Table with columns:
| Element | In Design | In Implementation | Status |
Status: ✅ Present | ❌ Missing | ⚠️ Different

### 3. Quantitative Comparison (DOM/CSS Inspection)
Table with columns:
| Property | Element | Expected | Actual | Status | Diff |
Properties: color (hex), font-size (px), spacing/margin/padding (px), border-radius (px), width/height.
Status: ✅ Match | ❌ Mismatch
Tolerance: 2px for spacing, exact match for colors (configurable in config).

### 4. Qualitative Comparison (AI Vision)
Bullet list of observations:
- Layout structure match/mismatch
- Visual hierarchy assessment
- Content correctness
- Responsive behavior notes

### 5. Missing from Spec
Elements present in the design but not documented in the feature file.

### 6. Dynamic Content Regions
Regions marked as dynamic (excluded from comparison): user avatar, timestamps, dynamic counters, etc.

### 7. Recommendation
AI-generated summary: pass, fail with list of fixes needed, or partial with caveats.

## Comparison Methodology

| Check Type | Tool | What It Measures |
|------------|------|-----------------|
| Qualitative | AI Vision | Layout, hierarchy, element presence, content |
| Quantitative | DOM/CSS inspection via Playwright | Colors, spacing, fonts, sizes |

## Tolerance Configuration

Read from `.argus/config.yaml` under `uiVerification`:

| Setting | Config Key | Default |
|---------|------------|---------|
| Spacing tolerance | `uiVerification.spacingTolerance` | 2px |
| Color match | `uiVerification.colorMatch` | exact |
| Font size tolerance | `uiVerification.fontSizeTolerance` | 1px |
